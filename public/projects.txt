descriptions : eTuitionBD
  eTuitionBD is a comprehensive online tuition management platform that connects students with qualified tutors
  in Bangladesh. The platform streamlines the process of finding, applying for, and managing tuition
  opportunities with automated workflows, digital tracking, and transparent payment systems.

  TECH_STACK
   - Frontend: React 19, Vite, Tailwind CSS, DaisyUI
   - Backend: Node.js, Express, MongoDB with Mongoose
   - Authentication: Firebase, JWT
   - Payments: Stripe
   - Styling: Tailwind CSS + DaisyUI
   - Other: React Router DOM, Axios, Framer Motion, AOS animations

  CHALLENGES
   - Implementing role-based access control for students, tutors, and admin users with different dashboard
     functionalities
   - Integrating Stripe payment system for tuition transactions with proper security measures
   - Managing real-time application status updates between tutors and students
   - Handling serverless database connections efficiently to prevent multiple connections in Vercel environment
   - Creating a comprehensive dashboard system with different views for each user role

  IMPROVEMENTS
   - Planning to enhance the matching algorithm between tutors and tuition requirements
   - Adding video calling integration for online tutoring sessions
   - Implementing a rating and review system for both tutors and students
   - Adding more advanced filtering and search capabilities for tuition posts
   - Expanding to support more subjects and educational levels
   - Adding notification systems for application updates and payment confirmations
]

     importing goods)
